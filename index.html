especialidade

HTML

HTML é uma linguagem de marcação de hipertexto. Ou seja a linguagem de marcação diz para o servidor da web qual é o estilo de e a estrutura do documento, já o hipertexto nada mais é do que um texto que faz referencia a outros textos, por fim o HTML não é considerado uma linguagem de programação pois com ele você não consegue criar funcionalidades complexas porem com o HTML você pode criar a estrutura de um site com seções links, imagens, carrossel, parágrafos, menu e etc..

CSS

CSS é utilizado para estilização no meu caso ele é utilizado para estilizar o HTML portanto o com CSS que consigo mudar cor de fundo, cor da fonte, espaçamentos e etc.. basicamente você consegue mudar tudo visual mente do seu site com o CSS assim deixando ele mais bonito.

JavaScript

JavaScript é uma linguagem de programação que permite que você faça muitas funcionalidades complexas como calcular, atualizar conteúdos de tempos em tempos etc.. JavaScript é muito utilizado na programação WEB
